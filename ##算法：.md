##算法：
**时间复杂度：**一般是指影响总操作数最大的因素；
O(1)<O(lgn)<O(n)<O(nlog2n)<O(n*2)<O(n*a)<O(2*n)<n!

**四大基本算法**
1.分治法：把一个问题拆分成多个小规模的相同子问题，每个子规模之间是相互独立的，一般用地柜解决，也可以用循环。**关键词：一分为二，一般用递归解决**
经典问题：归并排序、快速排序、矩阵乘法、二分查找、斐波那契数列等
2.贪心法：局部最优，但整体不见得最优。每步有明确的。既定的策略。
经典问题：最小生成树问题（普利姆算法、库里斯卡尔算法）、背包问题、活动场地问题

3.动态规划法：关键词**最优子结构、动态规划、子模式，自顶向下时间复杂度：O（2^n）、自底向上时间复杂度为O（n^a）（空间复杂度为：O（n）或者O（n^2）



**直接插入排序：**
首次把第一个当做有序区，每次用有序区的最后一个元素去和无序区的元素比较，如果比它小依次在有序区中从后往前比较，并且移动元素。

快速排序：
定义收尾指针，任取第一个元素，左边的小于等于枢轴，右边大于等于枢轴，从后往前走，确保数据不丢失，递归层数：O（lg2n）、每层划分：O（n），**最好情况下时间复杂度：O（nlg2n）、空间复杂度：O（lg2n）**；**最坏情况：**递归层数：O（n）、每层划分：O（n），**时间复杂度：O（n^2），空间复杂度：O（n)**,**平均时间复杂度：就是最好时间复杂度，如果元素都具有相同值时，运行时间复杂度为最坏情况。**
/#include <stdio.h>

\#include <stdlib.h>

\#define N 10

// 快速排序函数

int quick_sort(int *a, int low, int high) {

int i = low; // 第一位

int j = high; // 最后一位

int key = a[i]; // 将第一个数作为基准值


while (i < j) {

while (i < j && a[j] >= key) {j--;}

a[i] = a[j];

while (i < j && a[i] <= key) {i++;}

a[j] = a[i];}

a[i] = key;

if (j - 1 > low) {quick_sort(a, low, i - 1);}

if (i + 1 < high) {quick_sort(a, i + 1, high);}

return 0;

}

// 主函数

for (i = 0; i < N; i++) {printf("%d ", a[i]);}

// 调用快速排序

quick_sort(a, 0, N - 1);

printf("排序之后:\n");

for (i = 0; i < N; i++) {printf("%d ", a[i]);}

}

##最大子序和问题（贪心算法）
###### 就是从给定的整数序列中找到连续的一串整数（至少包含一个整数），要求这串整数和是所有整数串中最大的。

**该算法的解题思路：从头哦遍历整个序列，对于每一个元素，如果它的存在有助于找到最大自序和（和大于0），就把它加入整形串中，反之则放弃当前元素，继续从下一个元素开始找。
取一个元素，判断它们的和是否大于0，如果大就保留，如果小于就舍弃，并且每次都记录他们的和的值，当遍历完整个数组，找到和最大的终结数组里面的数就行。**
